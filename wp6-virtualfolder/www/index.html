CCP4 suite

The CCP4 (Collaborative Computational Project, Number 4) software suite is a collection of programs and associated data and software libraries which can be used for macromolecular structure determination by X-ray crystallography.

To enable local copy of CCP4 suite you agree that you have Academic or Commercial License. If not, please obtain a license first at CCP4License.

Scipion

Scipion is an image processing framework to obtain 3D models of macromolecular complexes using Electron Microscopy.

To enable and start local copy of Scipion Webtools, please click the Enable button.
